이 블로그는 **프레임워크 없이** 순수 HTML, CSS, JavaScript만으로 만들어졌습니다.

## 어떻게 동작하나요?

`posts/` 폴더에 마크다운 파일을 추가하고, `posts/index.json`에 메타데이터 한 줄을 더하면 새 글이 목록에 나타납니다.

지원하는 문법은 다음과 같습니다.

- 제목 (`#` ~ `######`)
- **굵게**, *기울임*, `인라인 코드`
- 순서 있는/없는 목록
- 인용문
- 링크와 이미지
- 코드 블록

```js
function hello() {
  console.log("Hello, World!");
}
```

> 이것은 인용문입니다.

1. 첫 번째
2. 두 번째
3. 세 번째
